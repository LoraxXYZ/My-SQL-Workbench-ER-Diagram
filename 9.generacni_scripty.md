# 9. generační scripty

Kompletní skript slouží k vytvoření databáze pomocí SQL příkazů. Obsahuje definice tabulek, primárních klíčů a cizích klíčů.

## Založení tabulky Team (týmy)

Základní tabulka bez vazeb.

CREATE TABLE IF NOT EXISTS `mydb`.`Team` (
  `id_team` INT UNSIGNED NOT NULL AUTO_INCREMENT,
  `name` VARCHAR(45) NOT NULL,
  PRIMARY KEY (`id_team`)
) ENGINE = InnoDB;

AUTO_INCREMENT = ID se generuje automaticky
NOT NULL = název týmu musí být vyplněn

## Založení tabulky Match (zápasy + cizí klíče)

Tady už používáme FOREIGN KEY – vazby na tabulku Team.

CREATE TABLE IF NOT EXISTS `mydb`.`Match` (
  `id_match` INT UNSIGNED NOT NULL AUTO_INCREMENT,
  `match_date` DATETIME NOT NULL,
  `home_team_id` INT UNSIGNED NOT NULL,
  `away_team_id` INT UNSIGNED NOT NULL,
  PRIMARY KEY (`id_match`),

  CONSTRAINT `fk_match_home_team`
    FOREIGN KEY (`home_team_id`)
    REFERENCES `mydb`.`Team` (`id_team`)
    ON DELETE NO ACTION
    ON UPDATE NO ACTION,

  CONSTRAINT `fk_match_away_team`
    FOREIGN KEY (`away_team_id`)
    REFERENCES `mydb`.`Team` (`id_team`)
    ON DELETE NO ACTION
    ON UPDATE NO ACTION
) ENGINE = InnoDB;

Každý zápas má:

* domácí tým
* hostující tým

oba jsou propojené přes cizí klíče

## Založení tabulky Customer (zákazníci)
CREATE TABLE IF NOT EXISTS `mydb`.`Customer` (
  `id_customer` INT UNSIGNED NOT NULL AUTO_INCREMENT,
  `name` VARCHAR(100) NOT NULL,
  `email` VARCHAR(100) NULL,
  PRIMARY KEY (`id_customer`)
) ENGINE = InnoDB;

email může být NULL (není povinný)

## Založení tabulky Ticket (vstupenky + vazby)

Tady se propojuje zákazník a zápas.

CREATE TABLE IF NOT EXISTS `mydb`.`Ticket` (
  `id_ticket` INT UNSIGNED NOT NULL AUTO_INCREMENT,
  `seat` VARCHAR(100) NOT NULL,
  `price` DECIMAL(10,2) NOT NULL,
  `id_customer` INT UNSIGNED NOT NULL,
  `id_match` INT UNSIGNED NOT NULL,
  PRIMARY KEY (`id_ticket`),

  CONSTRAINT `fk_ticket_customer`
    FOREIGN KEY (`id_customer`)
    REFERENCES `mydb`.`Customer` (`id_customer`)
    ON DELETE NO ACTION
    ON UPDATE NO ACTION,

  CONSTRAINT `fk_ticket_match`
    FOREIGN KEY (`id_match`)
    REFERENCES `mydb`.`Match` (`id_match`)
    ON DELETE NO ACTION
    ON UPDATE NO ACTION
) ENGINE = InnoDB;

### Jaké vztahy jsme vytvořily

* Team → Match

1:N (tým hraje více zápasů)

* Match → Ticket

1:N (zápas má více vstupenek)

* Customer → Ticket

1:N (zákazník má více vstupenek)
