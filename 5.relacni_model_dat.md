# 5. Navržené tabulky, jejich datové typy a použitá omezení atributů.

Tabulka: Team

## Záznamy o fotbalových týmech.

Název	Datový typ	Modifikátory	Popis
|---|---|---|---|
team_id	INT	PK, NN, UN, AI	Unikátní identifikátor týmu
name	VARCHAR(100)	NN	Název týmu
Tabulka: Customer

## Záznamy o zákaznících.

Název	Datový typ	Modifikátory	Popis
|---|---|---|---|
customer_id	INT	PK, NN, UN, AI	Unikátní identifikátor zákazníka
name	VARCHAR(100)	NN	Jméno zákazníka
email	VARCHAR(100)		Email zákazníka
Tabulka: Match

## Záznamy o fotbalových zápasech.

Název	Datový typ	Modifikátory	Popis
|---|---|---|---|
match_id	INT	PK, NN, UN, AI	Unikátní identifikátor zápasu
match_date	DATETIME	NN	Datum a čas zápasu
home_team_id	INT	FK, NN	Domácí tým (odkaz na Team)
away_team_id	INT	FK, NN	Hostující tým (odkaz na Team)
Tabulka: Ticket

## Záznamy o prodaných vstupenkách.

Název	Datový typ	Modifikátory	Popis
|---|---|---|---|
ticket_id	INT	PK, NN, UN, AI	Unikátní identifikátor vstupenky
seat	VARCHAR(100)	NN	Číslo sedadla
price	DECIMAL(10,2)	NN	Cena vstupenky
match_id	INT	FK, NN	Odkaz na zápas (Match)
customer_id	INT	FK, NN	Odkaz na zákazníka (Customer)

### Legenda modifikátorů  
**PK** (Primary Key) - Primární klíč: Jednoznačný identifikátor záznamu v tabulce. 
**FK** (Foreign Key) - Cizí klíč: Odkaz na primární klíč do jiné tabulky (slouží pro tvorbu relačních vazeb). 
**NN** (Not Null) - Hodnota nesmí být prázdná, je povinná. 
**AI** (Auto Increment) - Automatické navyšování ID pro každý nový záznam databází.
